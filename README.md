# Tsovic-Tsov
Tsovic-Tsov — это интерактивный гид по Еревану, превращающий прогулки в культурное приключение. Сканируйте QR-коды на памятниках, чтобы собирать цифровые коллекции и узнавать историю города через увлекательные факты и визуальные материалы. Исследуйте улицы, дополняя физический мир цифровыми артефактами.
